❌ SI NO VES TOAST MESSAGES:
   - Verifica que los permisos estén habilitados
   - Reinicia la app

❌ SI LOGS MUESTRAN ERROR:
   1. "❌ SERVIDOR NO DISPONIBLE"
      → Probar en navegador: http://3.21.53.102:8080/api/products
      → Si no funciona, el servidor EC2 está apagado

   2. "❌ Error en API"
      → Verificar internet del emulador/dispositivo
      → Revisar AndroidManifest.xml permisos

   3. Connection refused / Timeout
      → IP incorrecta o servidor apagado
      → Verificar firewall/security groups

❌ SI PRODUCTO NO APARECE EN POSTMAN:
   - Se guardó localmente pero no en servidor
   - Revisar logs específicos del error
   - Verificar formato JSON del request
